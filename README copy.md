# Modern Portfolio Website

## Overview
This is a modern, responsive portfolio website template designed for developers and designers to showcase their work. The website features a clean and professional design with smooth animations and interactive elements.

## Features
- Fully responsive design that works on all devices
- Modern and clean UI with smooth animations
- Interactive project filtering
- Skills section with progress bars
- Contact form with validation
- Smooth scrolling navigation
- Cross-browser compatible

## Technologies Used
- HTML5
- CSS3 (with custom properties and flexbox/grid layouts)
- JavaScript (vanilla, no frameworks)
- Font Awesome for icons
- Google Fonts

## How to Use

### Basic Setup
1. Clone or download this repository
2. Open the `index.html` file in your browser to view the website
3. Customize the content to make it your own

### Customization

#### Personal Information
- Update your name, title, and description in the hero section
- Replace the placeholder image in the About section with your photo
- Update your personal details (name, email, location, education)
- Add your own social media links

#### Projects
- Replace the placeholder project images with screenshots of your own projects
- Update project titles, descriptions, and technologies used
- Add or remove project categories as needed
- Update the project links to point to your live demos and repositories

#### Skills
- Adjust the skill percentages to reflect your proficiency levels
- Add or remove skills as needed
- Customize the skill categories to match your expertise

#### Contact Information
- Update your email, phone, and location
- Connect the contact form to your preferred form handling service

### Styling Customization
- The website uses CSS custom properties (variables) for easy color customization
- Open `styles.css` and modify the root variables at the top of the file to change the color scheme

```css
:root {
    --primary-color: #6c63ff;     /* Main accent color */
    --secondary-color: #f50057;    /* Secondary accent color */
    --dark-color: #2c2c2c;         /* Text and dark elements */
    --light-color: #f4f4f4;        /* Background and light elements */
    --success-color: #28a745;      /* Success messages */
    --danger-color: #dc3545;       /* Error messages */
    --transition: all 0.3s ease;   /* Default transition */
    --box-shadow: 0 5px 15px rgba(0, 0, 0, 0.1); /* Default shadow */
    --border-radius: 8px;          /* Default border radius */
}
```
