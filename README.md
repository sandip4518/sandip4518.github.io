# Modern Portfolio Website

## Overview

This is a modern, responsive portfolio website template for developers and designers to showcase work. It uses plain HTML, CSS and JavaScript and is easy to customize and deploy as a static site.

## Features

- Fully responsive layout
- Clean UI with smooth animations
- Project gallery with filtering
- Skills section with progress indicators
- Contact form (front-end validation)
- Smooth scrolling navigation

## Technologies

- HTML5
- CSS3 (variables, Flexbox, Grid)
- Vanilla JavaScript
- Font Awesome (icons)
- Google Fonts

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your browser to preview, or run a simple local server (recommended)

### Run locally (recommended)

Use one of these quick methods to serve the site locally:

Using Python 3:

```bash
python -m http.server 8000
```

Using Node (http-server):

```bash
npx http-server -p 8000
```

Then open `http://localhost:8000` in your browser.

## File Structure

- `index.html` — main page
- `styles.css` — site styles
- `script.js` — interactive behaviour
- `src/` — optional assets or source files

## Customization

- Replace text, images and links in `index.html` to add your content
- Update color variables and layout in `styles.css`
- Replace placeholder project images and update project links

### Styling variables (example)

```css
:root {
  --primary-color: #6c63ff;
  --secondary-color: #f50057;
  --dark-color: #2c2c2c;
  --light-color: #f4f4f4;
  --transition: all 0.3s ease;
  --box-shadow: 0 5px 15px rgba(0, 0, 0, 0.1);
  --border-radius: 8px;
}
```
